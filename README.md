# Maincrafts-Internship-Task-3
Titanic Mini Exploratory Data Analysis (EDA)This repository contains Task 3 of the Data Science & Data Analysis with Python Internship. Building upon the gender and class analysis from Task 2 , this project performs a deeper, more advanced Mini Exploratory Data Analysis (EDA) on the Titanic passenger dataset.  The primary objectives are handling missing values efficiently, engineering new relational features, and crafting compelling data visualizations to tell a clear analytical story.  🚀 Features & Project RequirementsThe implementation satisfies all the core criteria outlined for the internship task:  Data Cleaning & Imputation   Handled missing values in the Age column by imputing them with the column mean.  Dropped highly sparse and irrelevant features (such as Cabin) to avoid noise.  Feature EngineeringAge Groups: Binned continuous ages into distinct categorical cohorts (Child, Teen, YoungAdult, Adult, Senior) using custom intervals.  Family Size: Synthesized a unified FamilySize metric by aggregating the sibling (SibSp) and parent/child (Parch) attributes.  Advanced Visualizations & Groupby Deep Dives   Calculated precise survival rate variations utilizing Pandas groupby mechanics.  Generated an Age Distribution Histogram to capture demographic density.  Created a Correlation Heatmap to identify underlying linear dependencies among numerical features.  Plotted a Survival by Family Size Bar Chart to observe risk trends based on family units.  📂 Project StructurePlaintext├── titanic.csv              # The raw Titanic dataset
