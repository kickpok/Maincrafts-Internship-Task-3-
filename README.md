This repository presents Task 3 of the Data Science & Data Analysis with Python Internship, where the focus shifts from the basic gender and class analysis of Task 2 to a more advanced Mini Exploratory Data Analysis (EDA) on the Titanic dataset. The project emphasizes efficient handling of missing values, thoughtful feature engineering, and the creation of clear visualizations to uncover meaningful insights. Missing ages were imputed using the column mean, while sparse and less relevant attributes such as Cabin were removed to reduce noise. New relational features were introduced, including age cohorts (Child, Teen, YoungAdult, Adult, Senior) and a synthesized FamilySize metric derived from sibling and parent/child counts. Analytical depth was added through groupby operations to study survival rate variations, complemented by visualizations such as an age distribution histogram, a correlation heatmap, and a survival‑by‑family‑size bar chart. These analyses highlight key findings: most passengers were concentrated between ages 20 and 40, survival chances were higher for small family groups compared to solo travelers or very large families, and expected correlations emerged between ticket fare, passenger class, and survival probabilities. The project is structured with the raw dataset, a Jupyter Notebook containing code and outputs, and documentation in the README. It can be run locally or on platforms like Google Colab with Python and libraries including pandas, seaborn, and matplotlib. Overall, this task demonstrates how careful data cleaning, feature engineering, and visualization can transform raw data into a compelling analytical narrative.
